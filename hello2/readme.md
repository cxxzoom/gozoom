## package
``` 
1. 类似于其他语言的模块或者库的概念
2. 封装、模块化、代码重用
3. 单独编译
4. 每个包都对应一个独立的名字空间
5. 首字母大写即可导出
6. 每个包在解决依赖的前提下，以导入声明的顺序初始化，每个包只会被初始化一次
```
